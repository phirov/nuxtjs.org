      ava is a powerful JavaScript testing framework, mixed with jsdom, we can use them to do end-to-end testing easily for nuxt applications.

      npm install --save-dev ava jsdom

      Add test script to the package.json

      package.json

      // ...
"scripts": {
  "test": "ava",
}
// ...
